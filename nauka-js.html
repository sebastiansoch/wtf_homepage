JavaScript

JS - informacje podstawowe

Do osadzenia skryptu na stronie wykorzystuemu znacznik <script>. Skrypt możemy umieścić bezpośrednio w pliku HTML lub podlinkować plik zawierający rzeczony.
<script src="javascript.js"></script>

Z czego składa się JS

  • wartności
  • operacje porównania
  • literały
  • zmienne
  • funkcje
  • obiekty

Przydatne:

Logowanie do konsoli: console.log('wartość');

Backtick / backquote - prosty użycie zmiennych w ciągach:
console.log(`Nazywam się ${name} i mam ${age} lat`);

Rodzaje zmiennych

  • var - deklaruje zmienną o zakresie funkcji lub globalnym
  • let - deklaruje zmienną lokalną o zasięgu bloku
  • const - deklaruje nazwaną stałą tylko do odczytu, lokalną o zasięgu bloku

Typy proste w JS (primitives)

  • String - tekst, otaczamy ' lub " (zalecam ')
  • Number - liczba
  • Bigint - duża liczba ;) większa niż 253
  • boolean - true lub false
  • null - wartość celowo niezdefiniowana
  • undefined - brak definicji
  • *symbol - tworzy "unikalne opakowanie"
