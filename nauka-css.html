Box model

Box model składa się z elementu, paddingu (marginesu wewnętrznego), border (ramki), margin (marginesu zewnętrznego)
Skrócony zapis wielkości:

  • padding: top right bottom left;
  • border: width style color;
    • border-radius -zaokrąglanie nalożników, nie wymaga zdefiniuowania border
  • margin: top right bottom left;
    • margin: __px auto; -wyśrodkowanie elementów w rodzicu

Sposób ustalania wielkości elemetu definiujemy przy pomocy własności box-sizing. Pzyjmuje ona dwie wartości content-box oraz border-box. Dla box-sizing: content-box zmiana wielkości paddingu oraz border wpływa na zmianę całego boxu, natomiast box-sizing: border-box; przy analogicznej zmianie nie zmienia wielkości boxu a wielkość pola z zawartością

Pozycjonowanie elementów

  • position: static; -element jest umieszczony zgodnie z normalnym przepływem dokumentu. Właściwości top, right, bottom, left i z-index nie mają wpływu. To jest wartość domyślna.
  • position: relative; -element jest umieszczany zgodnie z normalnym przepływem dokumentu, a następnie przesuwany względem siebie na podstawie wartości top, right, bottom, left. Przesunięcie nie wpływa na położenie innych elementów.
  • position: absolute; -element jest usuwany z normalnego obiegu dokumentów i nie jest tworzone miejsce dla elementu w układzie strony. Element jest pozycjonowany względem najbliższego pozycjonowanego przodka (jeśli istnieje) lub początkowego bloku zawierającego. Jego ostateczna pozycja jest określana przez wartości top, right, bottom i left.
  • position: fixed; -element jest usuwany z normalnego obiegu dokumentów i nie jest tworzone miejsce dla elementu w układzie strony. Element jest umieszczony względem początkowego bloku zawierającego, którym jest viewport w przypadku mediów wizualnych. Jego ostateczna pozycja jest określana przez wartości top, right, bottom i left.
  • position: sticky; -element jest pozycjonowany zgodnie z normalnym przepływem dokumentu, a następnie przesunięty względem najbliższego przewijanego przodka i zawierającego blok (najbliższy przodek na poziomie bloku), w tym elementy związane z tabelą, w oparciu o wartości top, right, bottom, i left. Przesunięcie nie wpływa na położenie innych elementów.
  • float - element umieszczany jest po lewej lub prawej stronie jego kontenera, umożliwiając zawijanie tekstu i elementów śródliniowych wokół niego. Element jest usuwany z normalnego przepływu strony, ale nadal pozostaje częścią przepływu (w przeciwieństwie do pozycjonowania absolute).
  • clear - określa, czy element musi zostać przeniesiony poniżej (cleared) elementów typu float, które go poprzedzają. Właściwość clear dotyczy elementów pływających i niepływających.
  • z-index -ustawia element na wybranej warstwie
