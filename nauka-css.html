Zapis właściwości
selector {
  właściwość: wartość;
}

Aby podlinkowac arkusz styli CSS należy w sekcji head umieścić wpis <link href="sciezka do pliku.css" rel="stylesheet" type="text/css">

Selektory i regóły

  • #id - # (hash) - selektor id
  • .class - . (kropka) - selektor klasy
  • element
  • !important - kasuje inne regóły, nadpisuje własności

Specyficzność

Jeśli specyficzność jest równa - pierwszeństwo ma selektor "bliżej" elementu

( _ , _ ) , _ , _ , _
!important style="" #name .name element

Dziedziczenie

Zapis dziedziczenia
selector {
  wartość: inherit;
}

Dziedziczymy z bezpośredniego rodzica.
Przykładowo: a { color: inherit; } umożliwi nam odzieciczenie koloru dla linków z elementu nadrzędnego np section zamiast stosowania -webkit-link;
